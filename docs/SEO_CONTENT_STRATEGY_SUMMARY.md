## ✅ DO's and DON'Ts

### ✅ DO THIS (Good for SEO & UX)
- Use ONE H1 per page with primary keyword
- Use 3-5 H2s for main sections
- Use 2-4 H3s under each H2
- Write headings for humans first, SEO second
- Include keywords naturally (1-2% density)
- Use internal links with descriptive anchor text
- Keep headings specific and actionable
- Test all links before publishing

### ❌ DON'T DO THIS (Hurts SEO & UX)
- Multiple H1s on one page
- Keyword stuffing in headings
- Skip heading levels (H1→H3)
- Use generic headings ("Products", "More")
- Copy exact competitor content
- Ignore mobile responsiveness
- Break heading hierarchy
- Use clickbait-style headings
